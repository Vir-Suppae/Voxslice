# Voxslice
Voxslice is a platformer/puzzle game in the terminal where you explore and rotate 2D slices of a 3D world to try to escape.

# Contributing
If you contribute AI generated code, please test it before making a pull request
Also, you can check for things that I have marked as TODO using the syntax TODO[p0-3]: message where 0 is lowest priority and 3 is critical priority
